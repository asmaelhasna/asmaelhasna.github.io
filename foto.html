1. JPG
JPG adalah ekstensi file gambar yang sangat umum digunakan pada berbagai perangkat dan platform digital. JPG sebenarnya merupakan versi singkat dari JPEG, dibuat karena keterbatasan sistem operasi lama seperti MS-DOS yang hanya mendukung tiga huruf ekstensi file. Meskipun secara teknis tidak berbeda dari JPEG, JPG menjadi lebih populer karena lebih ringkas dan tetap dikenali oleh sebagian besar perangkat lunak pengolah gambar. Format JPG menggunakan kompresi lossy, yang artinya sebagian data gambar dihilangkan untuk memperkecil ukuran file, meskipun ini bisa menyebabkan sedikit penurunan kualitas. JPG sangat cocok digunakan untuk gambar seperti foto pemandangan, potret, atau media sosial karena mendukung warna hingga 24-bit (16 juta warna).
2. PNG
PNG adalah format gambar yang mendukung transparansi latar belakang dan kualitas tinggi tanpa kehilangan data (lossless). Format ini sangat ideal untuk 16 ikon, logo, dan gambar web yang memerlukan latar belakang transparan. PNG mempertahankan detail asli gambar meskipun ukuran file-nya lebih besar dibanding JPG. Kelebihan lainnya adalah kemampuannya untuk menampilkan teks dan garis dengan sangat tajam. PNG mendukung berbagai tingkat transparansi (alpha channel), menjadikannya pilihan utama untuk web desain modern.
3. ICO
ICO adalah format gambar yang digunakan khusus untuk ikon di sistem operasi Windows. Format ini mampu menyimpan beberapa ukuran ikon dalam satu file, sehingga dapat menyesuaikan tampilan sesuai resolusi layar. ICO mendukung transparansi dan biasanya digunakan dalam aplikasi dan shortcut desktop. Ukuran file biasanya kecil karena resolusi ikon yang terbatas, meskipun bisa berisi gambar resolusi tinggi. Meskipun jarang digunakan di luar konteks ikon, ICO tetap penting dalam pengembangan antarmuka pengguna.
4. HEIF
HEIF adalah format gambar modern yang dirancang untuk menyimpan gambar berkualitas tinggi dengan ukuran file yang jauh lebih kecil dibandingkan JPEG. Format ini menggunakan teknologi kompresi yang sama seperti video HEVC (High Efficiency Video Coding), sehingga efisien dalam penggunaan ruang penyimpanan. HEIF mendukung fitur-fitur canggih seperti transparansi, animasi, serta metadata tambahan seperti kedalaman gambar atau informasi burst shot. HEIF telah digunakan secara luas di perangkat Apple seperti iPhone dan iPad, serta mulai mendapatkan dukungan di Android dan browser modern. Meskipun belum seuniversal JPEG, HEIF menjanjikan efisiensi tinggi dengan tetap mempertahankan kualitas visual.
5. SVG
SVG adalah format gambar vektor berbasis XML yang bisa diskalakan tanpa kehilangan kualitas. SVG sangat cocok untuk ikon, logo, dan ilustrasi web yang harus tampil tajam di semua resolusi layar. Karena berbasis vektor, file SVG biasanya sangat kecil dan dapat diubah langsung melalui kode. SVG juga mendukung animasi dan interaktivitas dengan bantuan JavaScript dan CSS. Format ini merupakan standar untuk grafis web modern.
6. WebP
WebP adalah format gambar modern dari Google yang menawarkan ukuran file lebih kecil dengan kualitas tetap tinggi. WebP mendukung kompresi lossy dan lossless, serta transparansi dan animasi seperti gabungan dari JPG, PNG, dan GIF. Format ini sangat ideal untuk website karena mampu mempercepat waktu muat halaman. Banyak browser modern seperti Chrome dan Firefox sudah mendukung WebP. Dengan efisiensi tinggi, WebP perlahan menggantikan format lama dalam penggunaan web.
7. AVIF
AVIF adalah format gambar modern yang dirancang untuk memberikan kualitas tinggi dengan ukuran file yang sangat kecil. Format ini menggunakan kompresi berbasis codec AV1, yang dikenal sangat efisien dan mendukung kompresi lossy maupun lossless. AVIF mendukung transparansi, warna HDR, dan kedalaman warna tinggi, menjadikannya pesaing kuat untuk format seperti JPEG, PNG, dan WebP. AVIF sangat cocok digunakan di web karena ukurannya lebih kecil dibanding format lain dengan kualitas serupa, sehingga membantu mempercepat waktu muat halaman. Namun, meskipun dukungan untuk AVIF semakin luas di browser modern, beberapa perangkat lama mungkin belum sepenuhnya kompatibel.
8. BMP
BMP (Bitmap) adalah format gambar yang dikembangkan oleh Microsoft dan digunakan secara luas di sistem operasi Windows. Format ini menyimpan gambar tanpa kompresi atau dengan kompresi ringan, sehingga menghasilkan kualitas gambar yang sangat tinggi. Namun, ukuran file BMP cenderung besar karena tidak mengutamakan efisiensi penyimpanan. BMP tidak secara langsung mendukung transparansi, meskipun bisa dimodifikasi agar mendukungnya. Karena ukuran filenya yang besar, format ini tidak ideal untuk penggunaan web tetapi tetap digunakan untuk tujuan pengolahan gambar mentah.
9. RAW
RAW adalah format gambar mentah yang dihasilkan langsung dari sensor kamera digital tanpa banyak pemrosesan. Format ini menyimpan semua data gambar asli, memungkinkan fotografer melakukan penyuntingan dengan fleksibilitas tinggi. Karena tidak dikompresi atau hanya dikompresi lossless, ukuran file RAW sangat besar. RAW tidak digunakan untuk tampilan langsung di web, tetapi sangat penting dalam dunia fotografi profesional.
10. HEIC
HEIC adalah format gambar yang dikembangkan oleh Apple dan digunakan sebagai standar default di iPhone untuk menyimpan foto. HEIC menggunakan kompresi yang sangat efisien, memungkinkan gambar berkualitas tinggi dengan ukuran file yang kecil. Format ini mendukung transparansi, animasi, serta metadata lengkap. Meski sangat canggih, HEIC belum sepenuhnya kompatibel di semua sistem operasi dan browser. Namun, popularitasnya terus meningkat seiring penggunaan iOS.
